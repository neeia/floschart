# FlOSChart

## Tracks:

- Skill Requirements
- Items
- Quests
- Achievement Diaries
- Any Wiki Page

A node on the graph can have an icon, a title, a description, and a value.
For skills, the value is a number. For quests, the value is a boolean. For items, the value can be either, but defaults to a boolean.


## Run

```shell
yarn install
yarn dev
```
